General information about the project

The COVID-19 analysis project in Bosnia and Herzegovina is here to show off the COVID pandemic in the county. We look at the country as a whole, so there will not be segmentation for the administrative areas. The data is gathered from the website of Ministry of Civil Affairs of Bosnia and Herzegovina and Our World in Data. The specific details about the algorithms and technologies used in this project you can find in the API and About sections. One more thing to say before we start with the analysis is that the project is open-source and the content of the project you can find on Github
